HEXO写作

新建文档

$ hexo new "article title"

上述命令的结果是在/hexo/source/_posts路径下新建了一个article.md文件，然后就可以打开编辑器写作了。

文档格式

新建文档后打开article.md文件，里面已经有以下内容

这些内容是 MarkDown 文档在被解析为静态网页文件时的相关配置，这些配置参数一般位于文件中最上方以——分隔的区域。

  • title是文件名。也是网页显示的文章标题。
  • date是建文档的当地时区时间
  • tags可以用-开头为文章添加标签，如：
    -Markdown
    当文档被解析为静态网页后就可以看到效果

文章分类

  • Categories是文章的分类，和标签不一样的是，分类和目录一样是有层次性的，比如，JAVA/JVM

摘要

手动设置文章摘要，只需要写上摘要并在后面加上

# Blog
Java第一篇：Java基础
Java第二篇：面向对象一
