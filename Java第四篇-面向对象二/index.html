抽象类

  1. abstract class Animal //抽象类
    public abstract void eat() //抽象方法
  2. 抽象类不一定有抽象方法，有抽象方法一定是抽象类或接口
  3. 抽象类不能直接实例化，如
    Animal a = new Animal();是错的
    可以用子类多态进行实例化
    Animal a = new Cat();
    a.eat //子类的eat不是抽象方法
  4. 抽象类的子类要不是抽象的，要不就得重写所有抽象方法。
  5. 继承抽象类的时候抽象方法必须重写，为什么还要继承抽象类呢？
    其实抽象类里面的抽象方法相当于制订了规范，子类必须要写某些方法

Abstract修饰符

  • abstract和static不能共同修饰
    因为static修饰的方法可以用 类名.方法调用，但是抽象方法不能调用
  • abstract不能和final 共存
    因为abstract修饰方法要让子类强制重写，但是final不让子类重写
  • abstract不能和private共存
    因为private不让子类访问

接口

概述

狭义：指java中的interface
广义：对外提供规则的都是接口

特点

成员变量：只能是常量，并且是静态的公共的
默认修饰符：public static final

static修饰的成员变量：是类变量，在堆中。作用域和类相同
不用static修饰的成员变量：是实例变量，在栈中，作用域和对应实例相同
构造方法：接口中没有构造方法
成员方法：接口中成员方法都是抽象的
默认修饰符：public abstract
接口中不能定义非抽象方法

Java基本数据类型和引用数据类型传递区别

结论：

  • 一个方法不能修改一个基本数据类型的参数（数值型和布尔型）。

  • 上面两种传递都进行了值拷贝的过程。
  • Java本质都是按值调用

关于静态方法/变量，非静态方法/变量

定义：

静态方法：即类方法，在类之中，以static关键字申明的方法。
使用方法：直接使用类名.类方法名来进行调用。

非静态方法：即成员方法，没有static关键字申明。
使用方法：需要先创建类对象，使用类对象名.成员方法名来进行调用。

静态变量：即类变量，在类之中，方法之外，以static关键字申明的变量，它属于整个类所有，而不是某个对象所有，即被类的所有对象所共享。
使用方法：直接使用类名.类变量名来进行访问。

非静态变量：即成员变量，在类之中，方法之外。属于对象，有默认初始值
使用方法：需要先创建类对象，使用类对象名.成员变量名来进行调用

局部变量：在方法中，属于栈，随着方法调用产生，调用完毕消失。没有默认初始值

注意：

  • 静态方法不可以调用非静态方法或者非静态变量，因为这个时候被调用者还没有初始化。

正则表达式

概述

  • 用来匹配符合某个语法规则的字符串 的字符串（描述密码规则）
  • regex 即regular expression
  • String有方法matches(regex)来判断字符串是否符合正则表达式

    字符类

    [abc] a或者b或者c
    [^abc] 除了abc的任何字符
    [a-zA-Z] a到z或者A到Z
    [a-d[m-p]] a到d或者m到p
    [a-z&&[^bc]] a到z，除了b和c
    [a-z&&[^m-p]] a到z，除了m到p

    预定义字符类

    . 任意一个字符（点）
    \d 数字 [0-9]
    \D 非数字 [^0-9]
    \s 空白字符 [\t\n|x0B\f\r]
    \S 非空白字符
    \w 单词字符 [a-zA-Z_0-9]
    \W 非单词字符
  • 注意
    以上用的时候要多加一个\，因为要\本身就是转义字符，\才代表字符’\’
    String regex = “\\d”

    数量词

    • ？ 0或1次

      • =0多次

      • =1次

    • {n} 刚好n次
    • {n,} 至少n次
    • {n,m} n到m次
